민원신청이 어려우시다면 민원신청 따라하기의 설명을 따라해 보세요.

자격득실확인서
발급 선택 자격득실
내역 확인 증명서
발급

01. 먼저 로그인 하신 후, 민원여기요 > 개인민원 > 증명서 발급·확인 > 자격득실확인서 메뉴에 접속합니다.

01. 건강보험 자격득실 내역을 확인합니다. 내역을 확인 하신 후 [프린트 발급]이나 [팩스전송] 버튼을 클릭합니다.

프린트 발급

01. [프린트발급] 버튼 클릭 시 증명서 생성을 위한 팝업이 호출되고 증명서(PDF)가 생성됩니다.

02. 파일을 저장 후 비밀번호를 입력하여 확인할 수 있습니다. (비밀번호 - 개인 : 생년월일 6자리, 사업자 : 사업자등록번호)

팩스 전송

01. [프린트발급] 버튼 클릭 시 증명서 생성을 위한 팝업이 호출되고 증명서(PDF)가 생성됩니다.

02. 파일을 저장 후 비밀번호를 입력하여 확인할 수 있습니다. (비밀번호 - 개인 : 생년월일 6자리, 사업자 : 사업자등록번호)
